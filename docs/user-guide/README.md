## What is TinyBoards?

TinyBoards is a self-hosted community platform similar to Reddit. It's organized around **boards** (like subreddits) where users can share posts, have discussions, and build communities. Each TinyBoards instance is independently operated — the person who runs the server sets the rules.

### Key Concepts

- **Boards** — Topic-focused communities (like `/b/gaming` or `/b/cooking`). Each board has its own moderators, rules, and appearance.
- **Feed Posts** — Link shares, images, or text posts with voting. These appear in the board's feed section.
- **Threads** — Forum-style discussion threads without voting. These appear in the board's threads section.
- **Streams** — Custom feeds you create by combining boards and flair filters.
- **Flairs** — Colored tags on posts and users for categorization and identity.

## Getting Started

### 1. Create an Account

Visit the site and click **Register**. Depending on how the site is configured, you may:
- Register immediately (open registration)
- Need an invite code from an existing user
- Need to submit an application that admins review
- Find that registration is closed

### 2. Set Up Your Profile

After registering, visit **Settings** to customize your account:
- **Display name** — How your name appears on posts and comments
- **Avatar** — Your profile picture (you can crop it after uploading)
- **Banner** — A header image on your profile page
- **Bio** — A short description about yourself
- **Theme** — Choose a visual theme for the site

### 3. Join Some Boards

Browse the **Boards** directory to find communities that interest you. Click **Subscribe** to add a board to your home feed.

### 4. Start Participating

- **Upvote** content you find valuable
- **Comment** on posts to join discussions
- **Create a post** to share something with a board
- **Save** posts you want to revisit later
